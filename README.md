# Calculator

---

_Simple calculator app using Flutter SDK_

## Features

- ➕➖ Add/Substract/Multiply/Divide Operation ✖️➗
- ⬅️ ClearAll/Backspace Buttons 🔙
- ➕/➖ Change Sign Button
- ☀️ Light/Dark Mode 🌙
